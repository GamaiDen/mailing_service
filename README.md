# 📧 Сервис управления рассылками

Веб-приложение на Django для управления рассылками сообщений клиентам.

## Функциональность

### Часть 1. Разработка сервиса
- **Управление клиентами** — CRUD для получателей рассылки
- **Управление сообщениями** — CRUD для шаблонов писем
- **Управление рассылками** — CRUD с валидацией дат и статусами
- **Отправка сообщений** — через `send_mail()` с фиксацией попыток (Attempt)
- **Главная страница** — статистика: всего рассылок, активных, получателей

### Часть 2. Расширение функциональности
- **Регистрация и аутентификация** — вход/выход, регистрация пользователей
- **Права доступа** — пользователи видят свои данные, менеджеры — все
- **Статистика** — успешные/неуспешные попытки для пользователя
- **Кеширование** — Redis для главной страницы

## Технологии
- Python 3.12
- Django 6.0
- PostgreSQL
- Redis (django-redis)
- Bootstrap 5
